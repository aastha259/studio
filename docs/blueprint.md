# **App Name**: Bhartiya Swad

## Core Features:

- Immersive 3D Landing & Navigation: A dynamic, visually rich 3D homepage featuring floating food items to captivate users. Provides intuitive login and signup paths with smooth transitions for a seamless user experience.
- Role-Based Authentication: Secure authentication system supporting user login via email/password and Google, directing to a personalized food dashboard. Includes a separate login mechanism for administrators to access their content management panel.
- Personalized User Food Dashboard: The user's central hub, displaying trending and categorized food items. Incorporates an AI tool to suggest personalized food recommendations based on the user's past orders and preferences.
- Interactive Food Display: Each food item is presented on an interactive card, complete with its image, name, price, rating, an 'Add to Cart' button, and an engaging 3D hover animation for a delightful browsing experience.
- Robust Cart & Checkout System: Allows users to add, remove, and review items in their shopping cart, displaying the running total. Facilitates a secure and straightforward checkout process for placing orders, which are then stored in Firestore.
- Admin Food Management Panel: An administrative interface empowering content managers to effortlessly add, edit, or delete food items, manage associated images, set pricing and categories, and designate items as trending, all managed in Firestore.
- Smart Search & Filter System: Integrates a dynamic search bar for keyword-based food discovery and intuitive category filters, enabling users to efficiently navigate the diverse food catalog.

## Style Guidelines:

- Primary color: A vibrant, inviting saffron orange (#E55C0A) that evokes the warmth and richness of Indian cuisine, providing an energetic and appetizing feel.
- Background color: A subtle, warm off-white (#FCF3ED) that complements the primary hue while providing a clean and elegant canvas, enhancing readability in a light color scheme.
- Accent color: A rich, deep cranberry red (#C40A3A) that provides strong contrast and sophistication, used for important actions and visual highlights without overpowering the primary color.
- Headlines font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and somewhat playful aesthetic that complements the 3D interactive elements. Body text font: 'Inter' (sans-serif) for its neutrality, readability, and clean lines, ensuring clarity across various content types.
- Utilize modern, line-based or flat-style icons that are easily recognizable and culturally relevant to Indian food, ensuring they are integrated seamlessly into the 3D environment without visual clutter.
- Adopt a clean, spacious layout with a card-based structure for food items, optimizing for readability and visual appeal. Employ full-width sections for immersive 3D experiences, ensuring responsiveness across all device sizes.
- Implement fluid page transitions, interactive 3D element animations on hover and rotation, and subtle parallax scrolling effects to enhance the sense of depth and interactivity, contributing to an immersive user experience.
